# **App Name**: Ramadan Companion

## Core Features:

- Daily Dua Display: Show a specific dua (supplication) for each day of Ramadan.
- Categorized Duas: Allow users to browse duas based on categories such as 'for myself', 'for family', etc.
- User Dua Submission: Allow users to submit their own duas to share with others.
- Share Dua: Enable users to share duas via social media or messaging apps.

## Style Guidelines:

- Primary color: Gold (#D4AF37) to reflect the richness and importance of Ramadan.
- Background color: Dark navy blue (#0A192F) for a serene and contemplative feel.
- Accent color: Cream (#F5F5DC) to provide contrast and highlight important elements.
- Headline font: 'Amiri', serif, for titles and headings, providing an elegant, traditional feel.
- Body font: 'Cairo', sans-serif, for body text and UI elements, ensuring readability and a modern touch.
- Use relevant Islamic icons and symbols such as crescent moons and stars, incorporating them tastefully.
- A clean and spacious layout with a focus on readability, making the content accessible and easy to navigate.
